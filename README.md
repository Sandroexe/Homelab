# Homelab

Willkommen in meinem Homelab-Repository! Hier dokumentiere ich die Planung, Struktur und den Aufbau meines Heimnetzwerks.

## 🚀 Das Projekt
Ziel ist der Aufbau eines stabilen, performanten und zukunftssicheren Netzwerks für mein Zuhause. Die gesamte Infrastruktur wird zentral verwaltet und ist auf Skalierbarkeit ausgelegt.

## 🛠 Das System: UniFi Ökosystem
Das Herzstück und die Basis meines Netzwerks ist die **UniFi Dream Machine (UDM)** von Ubiquiti.

**Warum dieses System?**
* **Zentralisierung:** Routing, Switching und WLAN-Management in einer Oberfläche.
* **UniFi Protect:** Nahtlose Integration von Sicherheitskameras ohne Abo-Kosten.
* **Erweiterbarkeit:** Einfaches Hinzufügen von Access Points und Switches je nach Bedarf.
* **Professionelle Features:** VLANs, Firewall-Regeln und detailliertes Monitoring.

## 📂 Repository-Inhalt
In diesem Repository findest du:
* **Netzwerkpläne:** Visuelle Darstellung der Topologie und des Racks (erstellt mit draw.io).
* **Grundrisse:** Heatmaps und Platzierung von APs/Kameras (geplant im UniFi Design Center).
* **Dokumentation:** Details zur Verkabelung (CAT.7), den Dosen und der Hardware-Liste.

## 📝 Geplante Komponenten (Roadmap)
- [ ] Zentrale: **UniFi Dream Machine**
- [ ] WLAN: UniFi Access Points (U6 Series)
- [ ] Sicherheit: UniFi Protect Kameras (G4/G5 Series)
- [ ] Infrastruktur: 19" Rack, Patchpanel & strukturierte Verkabelung
